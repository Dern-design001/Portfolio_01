Portfolio API Endpoints

Your backend API is live! Use these endpoints to manage your portfolio data.

GET /api/profile

Retrieve profile data

PUT /api/profile

Update profile data

GET /api/projects

Get all projects

POST /api/projects

Create a new project

PUT /api/projects?id={id}

Update a project

DELETE /api/projects?id={id}

Delete a project

GET /api/ventures

Get all ventures

POST /api/ventures

Create a new venture

GET /api/milestones

Get all milestones

POST /api/milestones

Create a new milestone
